## Usage

### Changing Mode

```
d -> Drawing Mode
e -> Editing Mode
l -> Labeling Mode
```

### Drawing Mode

```
u -> Undo (remove last added edge)
c -> Clear (remove all edges)
```

### Labeling Mode
